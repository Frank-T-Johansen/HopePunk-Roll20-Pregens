# Hope//Punk Roll20 Pregens

Free community pregen pack for the Hope//Punk tabletop RPG, for use with the public Roll20 Hope//Punk character sheet.

This version uses a best-of-both-worlds importer:

```text
Name (Background) Journal names for easier player selection
v0.7 armour/cyberware trait selector fields
movement_speed import
optional handout import
selected portrait/default-token linking
```

Start with `ROLL20_IMPORT_GUIDE.md`.
